# Código Arquivado

Esta pasta contém código que foi removido do dashboard multi-tenant principal mas preservado para referência futura.

## Por que foi arquivado?

Após apresentação aos superiores em 2025-11-11, decidiu-se remover métricas específicas da AllpFit para tornar o dashboard mais genérico e aplicável a outros segmentos (educação, financeiro, etc.).

## O que está arquivado aqui?

### 1. Funil de Conversão
- Métricas específicas do processo de vendas AllpFit
- Etapas: Lead → Qualificação → Proposta → Fechamento

### 2. Métricas de Qualidade
- Análise de condição física
- Objetivos do cliente
- Análise de IA específica
- Sugestões de disparo
- Probabilidade de conversão

## Pode ser usado novamente?

Sim! Este código pode servir de referência para:
- Implementar análises customizadas por tenant no futuro
- Reativar funcionalidades específicas da AllpFit
- Servir de template para outros clientes que queiram análises similares
